class igraph.drawing.matplotlib.polygon.MatplotlibPolygonDrawer(ax)[source]¶

Bases: object

Class that is used to draw polygons in matplotlib.

The corner points of the polygon can be set by the C{points} property of the drawer, or passed at construction time. Most drawing methods in this class also have an extra C{points} argument that can be used to override the set of points in the C{points} property.

draw(points, corner_radius=0, **kwds)[source]¶

Draws a polygon to the associated axes.

@param points: the coordinates of the corners of the polygon,

in clockwise or counter-clockwise order, or C{None} if we are about to use the C{points} property of the class.

@param corner_radius: if zero, an ordinary polygon will be drawn.

If positive, the corners of the polygon will be rounded with the given radius.
